
# Configuration
The configuration parameters are:

* user - required, Total Connect Username
* password - required, Total Connect Password
* include_non_bypassable_zones - Specifies if non bypassable zones such as fire and police should be included, true/false, default false
* allow_disarming - Specifies if disarming is allowed from the ISY, true/false, default false
* refresh_auth_interval - Number of minutes between refreshing of the authentication token to the TotalConnect2 API, default 120
* zone_query_delay_ms - Number of milliseconds to delay when querying zone info, default 500
